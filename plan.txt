## 1) High-Level Architecture (Target)
### Services
- **Frontend:** React (Vite)
- **Backend API:** Node.js (Express or NestJS)
- **Database:** PostgreSQL (recommended)
- **Search:** Elasticsearch
- **Cache / Jobs (optional but strong):** Redis + BullMQ (or similar)
- **AI Service:** Python FastAPI (RAG + tool calling)
- **Images:** Object storage (S3/Cloudflare R2) + CDN

### Key integrations (portfolio-grade)
- Elasticsearch for product search + filters + autocomplete
- CDN for product images
- AI assistant that recommends products from your own catalog (RAG), and can add to cart (tool calling)

## 4) Minimum API Contract (Suggested)
### Core (Node)
- POST /auth/signup
- POST /auth/login
- GET /products
- GET /products/:id
- POST /cart/items
- PATCH /cart/items/:id
- DELETE /cart/items/:id
- POST /checkout
- GET /orders
- GET /orders/:id

### Admin (Node)
- POST /admin/products
- PUT /admin/products/:id
- DELETE /admin/products/:id
- POST /admin/products/:id/images

### Search (Node + Elasticsearch)
- GET /search?q=&category=&minPrice=&maxPrice=&sort=&page=

### AI (Python)
- POST /chat
  - input: user message + optional user/cart context
  - output: assistant message + recommended products + actions
